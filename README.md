别踩白块儿 - C 语言实现
一个使用 C 语言和 EasyX 图形库开发的经典 "别踩白块儿" 小游戏，代码简洁易懂，适合 C 语言初学者学习图形编程和游戏开发基础。
运行环境
操作系统: Windows 7/8/10/11
编译器: Visual Studio 2019/2022 (推荐)、Dev-C++、Code::Blocks
图形库: EasyX Graphics Library 2022 或更高版本
安装与运行
1. 安装 EasyX 图形库
访问 EasyX 官网 下载对应编译器版本的安装包
运行安装程序，选择你的编译器安装路径
安装完成后重启编译器
2. 编译运行
打开你的 C 语言编译器
创建一个新的空项目
添加一个.c源文件
将游戏代码复制粘贴到源文件中
编译并运行程序
游戏规则
游戏界面由 5 列 4 行的方块组成
每行有且仅有一个黑色方块，其余为白色方块
玩家只能点击最下面一行的黑色方块
点击正确的黑块得 1 分，所有方块会整体上移一行
顶部会生成新的一行方块
如果点击了白色方块，游戏立即结束
游戏结束后按任意键退出程序
